# PoseTracker

PoseTracker provides:

- real-time motion tracking
- uploaded video movement analysis
- reference movement comparison
- movement similarity scoring
- pose keypoints
- joint angles

Developers can create **reference movements** and compare user movements to them.

Example use cases:

- sports coaching apps
- technique analysis platforms
- physiotherapy monitoring tools
- dance training platforms
