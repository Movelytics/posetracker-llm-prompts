# PoseTracker

PoseTracker provides:

- real-time pose estimation
- uploaded video pose analysis
- image pose detection
- body keypoints
- joint angles
- skeleton overlays

It can be integrated using:

- iframe
- WebView
- URL parameters
- postMessage events
